# Android MVVM 起步脚手架

基于谷歌最新 Jetpack，MVVM设计模式的一套快速开发库，整合Okhttp+RxJava+Retrofit+Glide等主流模块，满足日常开发需求。使用该框架可以快速开发一个高质量、易维护的Android应用。

## 基于 MVVMHabit AndroidX 分支

### 更新了很多依赖版本，添加了一些依赖， 使用GitHub自动拉取最新版本的依赖库
### Lombok 注解简化代码
### 搭配swagger注解，方便对接后端
